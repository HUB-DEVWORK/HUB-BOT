# Changelog

Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
версии — [SemVer](https://semver.org/lang/ru/). Каждый релиз — тег `vX.Y.Z`;
описания генерируются автоматически из PR/коммитов при пуше тега.

## [1.2.0] — 2026-07-13

Полноценное управление из кабинета — везде появились кнопки удаления и отмена действий.

- 🗑 **Удаление там, где его не хватало**: тариф, пользователь, кампания — теперь удаляются
  прямо из кабинета (тариф с проданными подписками защищён — предложит деактивировать).
- 📱 **HWID-устройства**: кнопка **−1** к лимиту (была только +1) и **«Сбросить устройства»** —
  отвязать все привязки, чтобы клиент переподключился заново.
- ↩️ **Отмена действия**: после изменения баланса всплывает кнопка **«Отменить»** — вернёт как было
  в один тап (случайно накинули не ту сумму — откатили).
- 👤 Удаление пользователя убирает его и с панели Remnawave (panel-first), и локально
  (подписки/платежи/тикеты — каскадом); служебные аккаунты защищены.

## [1.1.0] — 2026-07-13

Надёжность установки на общем сервере и видимость установочных сбоев.

- 🔌 **Установщик сам разруливает конфликт портов 80/443**: если они заняты (напр.
  панель Remnawave на том же хосте) — стек поднимается без Caddy, web публикуется на
  `127.0.0.1`, печатается готовый nginx-сниппет. Больше никаких «address already in use».
- 🩺 **Сбои миграций/старта теперь видны на дашборде телеметрии** (`source=install`) —
  entrypoint репортит их одноразовым POST со скрабом секретов, не только в `docker logs`.
- 🛠 Исправлен `docker compose` на чистой установке (потерянный том `update-signals`),
  понятнее промпт про адрес API Remnawave.

## [1.0.0] — 2026-07-13

Первый публичный релиз VPN-HUB BOT.

- 🤖 Telegram-бот: продажи, триал, смена тарифа, баланс, рефералка, тикеты, HWID
- 🖥 Админ-кабинет (React) + полноценная **админка прямо в боте** (`/admin`)
- 📱 Mini App: 8 тем, оплата Stars/картой/криптой, подключение в 3 шага
- 💳 21 платёжный провайдер за одним ABC, авто-выдача по вебхуку + реконсилятор
- 🚚 Импорт из shopbot / Bedolaga / RemnaShop / 3x-ui
- 🧠 ИИ-поддержка, Device Guard, авто-техрежим, бэкапы, телеметрия ошибок с кодами
- 📖 Документация: <https://docs.vpn-hub.pro>
