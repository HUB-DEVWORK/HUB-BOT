# Project skills

Скилы этого проекта живут **в репозитории** (`.claude/skills/<name>/SKILL.md`) и коммитятся —
так они общие для всех, кто работает над BOT (и для Claude Code, и для тебя с другом).

Каждый скил — папка с `SKILL.md`: фронтматтер (`name`, `description` — по нему скил
подхватывается) + инструкция. Добавляешь новый — просто клади папку сюда и коммить.

Текущие:
- `add-payment-gateway/` — добавить платёжного провайдера в абстракцию платежей.
- `add-db-model/` — добавить модель БД + миграцию.

Общий рабочий контекст для AI — в корневом [CLAUDE.md](../../CLAUDE.md) и `docs/context/`.
