# С нуля до живого: деплой → настройка → продажи

Единый стек — **бот + мини-аппа + сайт-кабинет + админка + воркеры** — поднимается одной
командой и настраивается из веб-кабинета, без правки кода.

## 1. Развернуть (одна команда)

```bash
git clone https://github.com/bini69-oi/HUB-BOT.git && cd HUB-BOT
./scripts/install.sh
```

Спросит и подставит в `.env`:
- **токен бота** (@BotFather),
- **домен** для кабинета/мини-аппы (нужен HTTPS — Telegram Mini Apps без TLS не откроются;
  без домена поднимется на `http://IP`, но кнопка мини-аппы не появится),
- **URL + токен панели Remnawave** (Enter — встроенная мок-панель для теста).

Дальше сам сгенерит секреты, поднимет весь стек в Docker и напечатает
`https://<домен>/admin/` + пароль администратора.

## 2. Что связалось само (ничего не трогать)

- **Бот ↔ мини-аппа**: из домена авто-проставляются `SUBSCRIPTION_MINI_APP_URL=<домен>/app` и
  `CABINET_URL=<домен>` (env `WEB__PUBLIC_URL`) → в боте сразу появляется кнопка «📱 Приложение»
  и синяя menu-кнопка. Задашь URL вручную в кабинете — авто-значение не перезапишется.
- **Общий пользователь/подписка/баланс** у бота, мини-аппы и сайта (одна БД).
- **Админ-аккаунт**, дефолтное меню, шаблоны уведомлений, топики отчётов — засеиваются на старте.

## 3. Настроить в кабинете (`/admin/`) — чтобы продавать

Минимум для запуска продаж:
1. **Тарифы** (screen «Тарифы»): планы + сроки + цены.
2. **Платёжки** (screen «Платежи»): включить хотя бы один шлюз (ЮKassa/крипта) и/или Telegram Stars.
3. **Панель**: убедиться, что Remnawave отвечает (иначе выдача не пройдёт).

Опционально (всё — тумблерами, применяется без перезапуска):
- Пробный период (`TRIAL_ENABLED`, дни/трафик/устройства/цена), рефералка (`REFERRAL_*`),
  промокоды, распродажи, автопродление.
- Оформление: баннеры экранов (`/setbanner` в боте или в кабинете), лого/приветствие,
  режим меню `MAIN_MENU_MODE` (inline-кнопки или reply-клавиатура под вводом).
- `HIDE_SUBSCRIPTION_LINK`, `SHOW_TRAFFIC_USAGE` — соблюдаются во всех поверхностях сразу.

## 4. Проверить, что живо

- `/start` в боте → приветствие + кнопка «📱 Приложение» → мини-аппа открывается и показывает
  тарифы/подписку.
- Тестовая покупка → провижнинг на панели → «🔌 Подключить» отдаёт ссылку/импорт.
- Кабинет `/admin/` → «📈 Аналитика»: воронка, выручка, источники.

## Обновление / ручной деплой

- Обновить живой стек: `./scripts/update.sh` (git pull + пересборка + миграции + рестарт).
- Деплой на существующий VPS (systemd + nginx): `./scripts/deploy.sh [user@host]`.
- Здоровье: `https://<домен>/health` → `200`.
